# AI面相分析知识体系

## 一、面部特征指标（15个核心指标）

### 1. 额头
- **检测**：测量发际线到眉心的高度与宽度，观察饱满度
- **含义**：主早年运势（15-30岁）、智慧与事业格局
- **正面**：额头饱满开阔，天庭气象俱佳，早年运势亨通，思维格局开阔
- **提醒**：额头略窄，建议多积累沉淀，厚积薄发型选手，中年后渐入佳境

### 2. 眉毛
- **检测**：分析眉形（弯/直/剑）、浓密度、眉间距、眉毛长度
- **含义**：主兄弟宫、情志与人际关系
- **正面**：眉清目秀，性情温和有条理，人缘极佳，贵人运强
- **提醒**：眉毛略散，情绪易波动，建议培养定力，事缓则圆

### 3. 眼睛
- **检测**：眼型（杏眼/丹凤/桃花）、大小、神采、眼间距
- **含义**：主心性、智慧、桃花与贵人
- **正面**：目光有神，心思通透，洞察力强，是天生的决策者
- **提醒**：眼神略显柔和，容易心软，注意在关键时刻坚守立场

### 4. 鼻子
- **检测**：鼻梁高度、鼻翼宽度、鼻头形状与大小、山根高低
- **含义**：主财帛宫、中年运（40-50岁）、自我意识
- **正面**：鼻梁挺拔丰隆，财运根基扎实，中年后财源广进
- **提醒**：鼻翼偏薄，理财意识需加强，建议开源节流、稳健投资

### 5. 嘴巴
- **检测**：唇形、厚薄、嘴角弧度、口型大小
- **含义**：主食禄、口才、晚年福气与感情表达
- **正面**：唇形端正有棱，口才了得，食禄丰厚，晚年安乐
- **提醒**：嘴唇偏薄，言辞犀利易得罪人，建议多倾听少评判

### 6. 耳朵
- **检测**：耳朵大小、耳垂厚度、耳廓形状、位置高低
- **含义**：主肾气、寿命、少年运与福气根基
- **正面**：耳厚垂珠，根基深厚，天生有福之相，一生少灾少难
- **提醒**：耳朵偏小，先天肾气需养护，注意劳逸结合

### 7. 下巴
- **检测**：下巴形状（圆/方/尖）、长度、丰满度
- **含义**：主晚年运、不动产、下属缘
- **正面**：下巴圆润有肉，晚运安稳，子女孝顺，安享天年
- **提醒**：下巴略尖，晚年需提前规划养老，多置固定资产

### 8. 颧骨
- **检测**：颧骨高低、宽窄、是否有肉包裹
- **含义**：主权力、管理能力与中年运势
- **正面**：颧骨端正有肉，管理能力出众，适合带团队、掌实权
- **提醒**：颧骨高露，个性强势需注意人际柔和度，刚柔并济为上

### 9. 法令纹
- **检测**：法令纹深浅、长短、对称度、起始位置
- **含义**：主威信、事业成就与社会地位
- **正面**：法令纹清晰对称，威望日增，事业上令出必行
- **提醒**：法令纹过早出现，操劳过度的信号，记得给自己松绑

### 10. 人中
- **检测**：人中长度、深浅、宽窄、清晰度
- **含义**：主寿元、生育、精力与生命力
- **正面**：人中深长清晰，生命力旺盛，精力充沛，子女缘厚
- **提醒**：人中偏浅，注意养生保精，规律作息是最好的投资

### 11. 脸型
- **检测**：整体轮廓分类（国字/瓜子/圆/菱形/长形）
- **含义**：主性格底色与人生基调
- **正面**：面型方圆有致，性格稳重包容，是值得信赖的伙伴
- **提醒**：脸型棱角分明，个性鲜明但易树敌，广结善缘可化解

### 12. 三庭比例
- **检测**：上庭（发际-眉）、中庭（眉-鼻尖）、下庭（鼻尖-下巴）比例
- **含义**：三庭均等为佳，分别对应早/中/晚年运势
- **正面**：三庭匀称，一生运势平稳上扬，各阶段皆有发展
- **提醒**：三庭不均，人生节奏有快有慢，抓住优势时段重点发力

### 13. 五眼比例
- **检测**：面部宽度是否约等于五个眼睛宽度，眼间距是否合理
- **含义**：主面部协调度，影响人缘与第一印象
- **正面**：五眼比例协调，面部和谐大方，第一印象极佳
- **提醒**：比例略有偏差，独特个性的标志，用实力征服比外表更持久

### 14. 面部对称度
- **检测**：左右脸对称程度（眉、眼、嘴角、颧骨对称性）
- **含义**：对称度高主内心平和、运势稳定
- **正面**：面部高度对称，身心和谐，运势平稳顺遂
- **提醒**：左右略有差异（人人如此），说明内心丰富，多面性格是优势

### 15. 肤色气色
- **检测**：面部整体肤色、光泽度、气色红润度
- **含义**：主近期运势与健康状态，是最动态的指标
- **正面**：气色红润有光泽，近期运势上行，精气神俱佳
- **提醒**：气色偏暗，近期需注意休息调养，运势低潮期适合蛰伏蓄力

## 二、五维运势权重

| 运势维度 | 关联特征 |
|---------|---------|
| **事业运** | 额头、眉毛、颧骨、法令纹、三庭比例 |
| **财运** | 鼻子、嘴巴、耳垂、下巴、肤色气色 |
| **桃花运** | 眼睛、嘴巴、脸型、五眼比例、面部对称度 |
| **贵人运** | 眉毛、耳朵、眼睛、颧骨、人中 |
| **健康运** | 人中、耳朵、肤色气色、法令纹、面部对称度 |

> 评分逻辑：每个维度取关联特征的加权均分，主特征权重0.25，辅助特征权重0.15，总分映射到0-100分。

## 四、双人配对规则

### 匹配逻辑

**互补原则（60%权重）**：
- 额头饱满 × 下巴圆润 = 早年+晚年互补
- 眼大有神 × 鼻挺有力 = 感性+理性平衡
- 颧骨高 × 眉眼柔 = 强势+温和互补

**相合原则（40%权重）**：
- 脸型相近 → 审美契合度高
- 三庭比例接近 → 人生节奏同步
- 肤色气色均佳 → 当下运势共振

### 配对维度
- **默契指数**：五眼比例+脸型相似度
- **互补指数**：性格特征互补程度
- **运势共振**：五维运势的协同度
- **综合契合度**：加权总分，映射到百分制
