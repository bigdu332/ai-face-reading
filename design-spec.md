## 一、配色体系

### 主色

| 角色 | 色值 | 用途 |
|------|------|------|
| 深紫 | `#2D1B69` | 品牌主色，标题、按钮、核心交互 |
| 金色 | `#D4AF37` | 强调色，评分数字、图标高亮、装饰纹样 |
| 暗红 | `#8B2252` | 辅助强调，标签、警示、情绪渲染 |

### 背景色

| 角色 | 色值 | 用途 |
|------|------|------|
| 主背景 | `#0F0B1A` | 深色页面底色 |
| 卡片背景 | `#1A1230` | 内容卡片、弹窗底色 |
| 渐变背景 | `#0F0B1A → #2D1B69` | 页面纵向渐变 |

### 文字色

| 角色 | 色值 | 用途 |
|------|------|------|
| 主文字 | `#F5F0E8` | 正文、标题（暖白，避免纯白刺眼） |
| 次级文字 | `#A89EC2` | 说明文字、辅助信息 |
| 弱化文字 | `#6B5F8A` | 提示、水印、禁用态 |

### 辅助色

| 角色 | 色值 | 用途 |
|------|------|------|
| 成功/吉 | `#4CAF50` | 正向结果、高分指标 |
| 警告/中 | `#FF9800` | 中等评分、注意提示 |
| 分割线 | `#2A2045` | 卡片内分隔 |
| 遮罩层 | `#0F0B1A` 80% | 弹窗、加载蒙层 |

## 二、3种分享卡片风格

### a) 国潮风

- **配色**：深紫 `#2D1B69` 底 + 金色 `#D4AF37` 纹样 + 暗红 `#8B2252` 印章
- **布局**：竖版 750×1334。顶部祥云纹装饰带 → 用户面部圆形头像（金色描边） → 面相评语（竖排书法字体） → 底部印章式总评分 + 小程序码
- **调性关键词**：东方美学 · 宫廷感 · 墨韵留白 · 传统纹样

### b) 赛博朋克

- **配色**：纯黑 `#0A0A0A` 底 + 霓虹紫 `#B44DFF` + 青色 `#00F5D4` 描边光效
- **布局**：竖版 750×1334。故障艺术(glitch)标题 → 用户头像（扫描线 + 数据流叠加效果） → 面部特征雷达图（霓虹描边） → 底部评分数字（大字号 LED 风格） + 二维码
- **调性关键词**：未来感 · 数据可视化 · 霓虹光效 · 科技解码

### c) 简约高级

- **配色**：米白 `#FAF8F5` 底 + 深灰 `#2C2C2C` 文字 + 金色 `#D4AF37` 点缀
- **布局**：竖版 750×1334。大面积留白 → 用户头像（圆角矩形，细线描边） → 单行核心评语（衬线体，居中） → 3项关键指标（横向等分，极简图标） → 底部品牌 logo + 二维码
- **调性关键词**：克制留白 · 轻奢质感 · 高级灰调 · 无印风

## 三、关键页面布局

### 1. 首页

- **顶部**：品牌 logo + 标题「AI面相」，祥云动态背景
- **中部**：核心交互区 — 大圆形拍照/上传按钮（金色脉冲光环），下方文案引导「拍一张，看面相」
- **底部**：功能入口横排 — 「好友配对」「历史报告」「分享赚福气」，悬浮式胶囊按钮

### 2. 加载页（分析中）

- **全屏沉浸**：深紫渐变背景
- **中部**：用户上传的面部照片，叠加扫描光线动效（金色横线从上至下循环扫描）
- **下方**：分析进度文案逐条切换 —「正在解析面部轮廓…」「五官比例计算中…」「命理数据生成中…」
- **底部**：进度条（金色填充）

### 3. 结果页（总览）

- **顶部**：用户头像 + 面相称号（如「天庭饱满·福泽深厚」），金色装饰框
- **中部**：总评分（大字号数字跳动至最终值） + 五维雷达图（官禄、财运、情感、健康、人缘）
- **下方**：3条核心面相解读摘要（卡片式，左侧图标 + 右侧文字）
- **底部**：双按钮 —「查看详细报告」+「分享给好友」

### 4. 详细报告页

- **长页面滚动结构**：
  - 顶部总评区（复用结果页头像+评分）
  - 分区卡片依次展开：五官逐项分析（眉、眼、鼻、口、耳），每项含评分条 + 解读段落
  - 综合运势分析（今年/明年，带时间轴）
  - 改运建议（图标列表）
- **底部悬浮栏**：「保存报告」+「分享」

### 5. 配对页

- **顶部**：标题「面相缘分配对」
- **中部**：左右双头像布局（VS 对称结构），中间连线动效（红线牵缘）
- **下方**：配对契合度分数 + 各维度对比条形图（并列双色条）
- **底部**：「邀请TA来测」分享按钮

### 6. 分享页（生成卡片）

- **顶部**：风格切换标签栏（国潮 / 赛博朋克 / 简约）
- **中部**：实时预览区，展示当前风格的分享卡片
- **底部**：操作栏 —「保存图片」+「分享到微信」+ 「复制链接」

## 四、动效建议

### 加载扫描

面部照片上叠加一条金色光线，从上至下匀速扫描（1.5s/次），配合微弱粒子飘散。扫描线过处，面部区域短暂高亮，营造「AI正在识别」的仪式感。

### 数字跳动

评分数字从 0 弹跳递增至最终值，使用 ease-out 缓动（总时长 1.2s）。到达终值时触发一次金色光晕扩散（scale 0→1.5，opacity 1→0，0.4s）。

### 雷达图绘制

五维雷达图各轴从圆心向外延伸绘制，依次展开（每轴间隔 0.15s），配合轻微弹性回弹(overshoot)，总时长约 1s。

### 页面转场

页面间采用 **共享元素过渡**：头像从结果页平滑移动到报告页顶部（位移+缩放，0.4s ease-in-out）。其余内容从下方 fade-slide-up 进入（translateY 30px→0，opacity 0→1）。

### 配对连线

两头像就位后，中间生长出一条红色丝线（SVG path 动画 stroke-dashoffset），线到终点时爆发小型烟花粒子，契合度数字随即跳动出现。

### 微交互

- 按钮点击：scale 0.95→1 + 涟漪扩散（0.2s）
- 卡片风格切换：crossfade 渐变过渡（0.3s）
- 滚动视差：报告页背景纹样以 0.3× 速率跟随滚动，营造层次深度
